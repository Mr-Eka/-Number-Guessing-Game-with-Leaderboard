# 🎯 Number Guessing Game with Leaderboard

A desktop-based Number Guessing Game developed using Python, Tkinter, and SQLite. The application allows players to guess randomly generated numbers across multiple difficulty levels, track their performance, view leaderboards, and maintain player statistics.

## Features

* Multiple difficulty levels (Easy, Medium, Hard)
* Random number generation
* Real-time feedback (Too High / Too Low)
* Player profile tracking
* SQLite database integration
* Persistent leaderboard system
* Best score tracking
* Interactive Tkinter GUI
* Game reset functionality

## Technologies Used

* Python
* Tkinter
* SQLite3
* Random Module

## Database

The project uses SQLite to store:

* Player Name
* Number of Attempts
* Leaderboard Rankings

Database File:

scores.db

## Project Workflow

1. Enter player name.
2. Select difficulty level.
3. Start the game.
4. Guess the secret number.
5. Receive hints after each attempt.
6. Score is stored automatically upon winning.
7. View leaderboard and player statistics.
